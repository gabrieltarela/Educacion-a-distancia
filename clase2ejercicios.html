Clase 2: El Principio de Atomicidad (1FN)

Cómputo Real

🛒 1. El Carrito "Pegoteado"

Un sistema de ventas guarda los pedidos con grupos repetitivos en una sola celda:

ID: 101 | Cliente: Ana Silva | Productos: Laptop, Mouse, Teclado
  • Reto Técnico: El Monitor cuesta $200 y el Teclado $50. ¿Cómo puede la PC sumar el total si están pegados en la misma celda? [1],[2]
  • Acción: Reconstruye la tabla para que cumpla con la 1FN, eliminando los grupos repetitivos. [3],[4]
Admin Pro

📱 2. Contactos al Límite

La base de datos de proveedores tiene este problema de atomicidad en la columna de teléfonos:

Proveedor: Tech-Uy | Teléfonos: 099123456, 24001122, 098765432
  • El Conflicto: Necesitas enviar un SMS automático. ¿Cómo detecta el sistema el primer número sin confundirse con los demás? [4]
  • Acción: Aplica la restricción de Dominio Atómico para que cada celda sea un valor indivisible. [4],[5]
Data Master

🎮 3. Hobby-Games Challenge

En el club de juegos, la tabla de socios presenta el siguiente desorden informativo:

Socio: Diego | Categorías: Estrategia, Rol, Acción
  • El Problema: Queremos buscar solo a los socios de "Rol". ¿Por qué tener todo junto impide una búsqueda precisa del sistema? [4]
  • Acción: Transforma este caos en un diseño óptimo que permita el procesamiento automático de datos. [3],[2]
💡 RECUERDA: La 1FN prohíbe los grupos repetitivos. Cada celda debe ser un valor único e indivisible para el sistema informático. [4]
